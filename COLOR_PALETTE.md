# PARE EDU HUB - Color Palette

## Overview

Color palette yang didesain berdasarkan logo PARE EDU HUB dengan fokus pada warna emas (gold) dan navy blue yang mencerminkan kemewahan pendidikan dan profesionalisme.

## Light Theme Colors

### Primary Colors

- **Primary**: `oklch(0.7 0.15 60)` - Golden/Amber (warna utama dari logo)
- **Background**: `oklch(0.98 0.01 45)` - Warm white dengan hint emas
- **Foreground**: `oklch(0.2 0.05 240)` - Dark navy untuk teks

### Secondary Colors

- **Secondary**: `oklch(0.95 0.02 50)` - Light gold accent
- **Accent**: `oklch(0.92 0.03 55)` - Warm gold untuk highlights
- **Muted**: `oklch(0.96 0.01 240)` - Subtle navy tint

### UI Elements

- **Card**: Pure white `oklch(1 0 0)`
- **Border**: `oklch(0.9 0.01 240)` - Light navy border
- **Ring**: `oklch(0.7 0.15 60)` - Gold focus ring

## Dark Theme Colors

### Primary Colors

- **Primary**: `oklch(0.8 0.12 55)` - Bright gold untuk visibility
- **Background**: `oklch(0.15 0.02 240)` - Deep navy background
- **Foreground**: `oklch(0.95 0.01 50)` - Warm white text

### Secondary Colors

- **Secondary**: `oklch(0.25 0.03 240)` - Dark navy dengan warmth
- **Accent**: `oklch(0.3 0.03 240)` dengan foreground `oklch(0.85 0.08 55)`
- **Muted**: `oklch(0.25 0.02 240)` - Consistent navy base

### UI Elements

- **Card**: `oklch(0.18 0.02 240)` - Slightly lighter than background
- **Border**: `oklch(0.3 0.02 240)` - Subtle navy border
- **Ring**: `oklch(0.75 0.12 55)` - Bright gold focus ring

## Sidebar Theme

Menggunakan navy blue sebagai base dengan gold accents:

### Light Mode

- Background: `oklch(0.97 0.01 240)` - Very light navy
- Primary: `oklch(0.25 0.06 240)` - Deep navy
- Accent: `oklch(0.92 0.03 55)` - Warm gold

### Dark Mode

- Background: `oklch(0.12 0.02 240)` - Very deep navy
- Primary: `oklch(0.8 0.12 55)` - Bright gold
- Accent: `oklch(0.25 0.03 240)` - Navy accent

## Brand Inspiration

Warna-warna ini diambil dari:

- **Emas/Gold**: Melambangkan excellence, premium education, dan achievement
- **Navy Blue**: Melambangkan trust, professionalism, dan stability
- **Gradasi**: Menciptakan depth dan modern aesthetic

## Usage Guidelines

- Gunakan **Primary (Gold)** untuk CTA buttons, highlights, dan brand elements
- Gunakan **Navy** untuk text, borders, dan structural elements
- **Secondary** dan **Accent** untuk subtle highlights dan hover states
- Pastikan kontras yang cukup antara foreground dan background
- Chart colors sudah dioptimasi untuk accessibility dan readability

## Technical Notes

- Menggunakan OKLCH color space untuk better perceptual uniformity
- Lightness values diatur untuk optimal contrast ratios
- Chroma dan hue values konsisten untuk brand coherence
